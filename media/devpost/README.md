# Devpost Media Pack

All images are 1500x1000 PNGs, a 3:2 ratio suitable for Devpost gallery uploads.

Recommended upload order:

1. `01-cover.png` - hero image and project identity.
2. `02-architecture.png` - system architecture.
3. `03-spiral-memory-loop.png` - the memory breakthrough.
4. `04-qwen-cloud-integration.png` - Alibaba/Qwen integration proof.
5. `05-antipattern-repair.png` - negative outcomes become corrective pressure.
6. `06-governed-self-improvement.png` - self-improvement with guardrails.

Supporting assets:

- `07-video-storyboard.png` - quick video structure.
- `00-contact-sheet.png` - all gallery assets in one view.

Regenerate the media pack:

```bash
python3 scripts/generate-devpost-media.py
```
